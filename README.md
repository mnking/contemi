# Contemi Technical Notes
> ## Install Pegasus
###Clone wrap repository
```sh
git clone -b develop git@bitbucket.org:contemivn/pegasus-dashboard-docker-vagrant.git
```
####Run composer
```sh
./pegasus.sh composer -p workspaces/ -sv
```
####Run Docker
```sh
shell/docker/docker-run.sh -d
```
